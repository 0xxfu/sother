# Sother

## DEV

```shell
poetry install
```

```shell
make local
```

print default markdown:
```shell
sother ./solidity --print markdown
```
print secure3 markdown:
```shell
sother ./solidity --print secure3
```

Update slither

```shell
poetry update slither-analyzer
```

```shell
sother file.sol --print slithir
```

## Install to local

```shell
pip install .
```
